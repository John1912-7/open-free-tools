# Owned Channels

This file tracks channels that Open Free Tools controls or can safely automate through official project access.

## Status Legend

- `official_auto`: safe to publish automatically through official project access.
- `owned_manual`: owned channel, but publishing is manual until official automation is configured.
- `manual_or_later`: prepare content only; do not publish automatically.
- `blocked`: do not use until the blocker is resolved.

## Account Setup Checklist

- Create project profile with Open Free Tools name.
- Add hub URL.
- Add GitHub URL.
- Add short bio: "Free open-source browser tools for music, transcription, and creative workflows."
- Use the same logo/avatar across owned channels.
- Do not post aggressively from a brand-new account.
- Keep credentials outside the repository.
