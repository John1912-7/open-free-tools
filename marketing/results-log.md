# Promotion Results Log

Use this file for weekly promotion summaries.

## Metrics to Track

| Metric | Source | Why it matters |
| --- | --- | --- |
| Visits by UTM source | Google Analytics | Shows which owned channels bring traffic. |
| Tool page clicks | Google Analytics | Shows whether visitors try a tool. |
| GitHub stars | GitHub | Measures open-source interest. |
| GitHub issues | GitHub | Measures useful feedback. |
| Bad conversion reports | GitHub issue template | Helps improve Audio-to-MIDI quality. |
| MIDI downloads | app analytics when available | Shows whether conversion results are useful. |
| Open-in-trainer clicks | app analytics when available | Shows complete workflow adoption. |
