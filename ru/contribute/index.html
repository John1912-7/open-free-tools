Правила проекта

Core-функции остаются бесплатными и open-source. Проект не копирует бренды, логотипы, UI, ассеты, private API или защищенное поведение платных сервисов.

Как проект может себя поддерживать

Проект может использовать донаты, sponsors, продвижение своих инструментов и уважительную рекламу на контентных страницах. Реклама не должна мешать рабочему интерфейсу инструментов.
