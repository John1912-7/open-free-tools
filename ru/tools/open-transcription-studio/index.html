Что это будет

Будущий open-source редактор транскрипций для аудио и видео с privacy-first подходом и экспортом субтитров.

Планируемые функции

  • Загрузка аудио/видео
  • Редактор текста
  • Таймкоды
  • Экспорт TXT/SRT/VTT
  • Локальное сохранение проекта

Следующее

  • Описать MVP
  • Выбрать local-first подход
  • Не копировать бренд и UI платных сервисов
  • Сделать отдельный проект
